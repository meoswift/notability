# notability - Jumbo Notes

Basically, a Google Keep clone made from scratch :-)
(using HTML, CSS, JS)

Working functionalities:
1. Edit notes (by clicking on the note)
2. Pin/Unpin notes
3. Archive/Unarchive notes
4. Delete notes
